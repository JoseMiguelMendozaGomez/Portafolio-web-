Nutri- IA

Este proyecto fue realizado como semestral para la materia de desarrollo #9. Consistía en una pagina donde podias hacer creación y logueo de usuarios, una vez ingresado podias solicitar recomendaciones de comidas saludables con ingredientes que tuvieras a tu disposición y esta te devolvia una respuesta utilizando la API de chatgpt.

Herramientas utilizadas:
Js+Express, React y Atlas MongoDB

Calculadora Edad

Esta aplicacion fue realizada como proyecto semestral de desarrollo #6. Esta consistia en una serie de preguntas que se le realizaban al usuario y dependiendo de las respuestas y segun su edad te daba la edad aparentada si esta era menor a la actual te daba consejos para mantener el estilo de vida y si era mayor te daba consejos para mejorar. Tambien se utilizo la API de chatgpt para las respuestas al usuario.

Herramientas utilizadas:
Java en android
